1.Bootstrap 5 มีอะไรใหม่ๆบ้างที่แตกต่างจาก Version เก่าอย่างน้อย 5 ตัวอย่าง

1. โลโก้ใหม่

2. คอมโพเนนต์ใหม่ ที่ชื่อว่า offcanvas.

3. ปรับปรุงและเพิ่มคอมโพเนนต์ของ forms

4. Browser ที่รองรับ

5. JavaScript.

2.จุดเด่นของ Bootstrap คืออะไรจงอธิบาย

มี UI เริ่มต้นแบบที่สวยงามและใช้งานง่าย

เป็นที่นิยมของนักพัฒนาทั่วโลก ทำให้สามารถเรียนรู้และแก้ปัญหาได้ง่าย

โค๊ดหรือชุดคำสั่งต่าง ๆ ค่อนข้างสะอาดมีโฟลเด้อต้นแบบแค่ 3 ส่วนคือ js, css, fonts

ประหยัดเวลาในการพัฒนาเว็บไซต์และนำไปพัฒนาต่อได้ง่าย

เป็น Responsive Framework พัฒนาเว็บไซต์ที่รองรับการแสดงผลได้หลากหลาย Device

3.จงอธิบายการติดตั้งและใช้งานแบบ CDN และแบบ Internal (Compiled CSS and JS) แตกต่างกัน อย่างไร

แบบ CDN ก็นำเข้าลิ้งเวอร์ชั้นที่ต้องการใช้ ถ้าเป็น JavaScript ใช้ script

แบบ Internal สร้างไฟล์ที่ชื่อว่า main.scss จากนั้นลิ้งไฟล์กับ index พิมพ์ @import "../node_modules/bootstrap/scss/bootstrap" เสร็จแล้วดาวน์โหลด scss live กด watch live จากนั้นเปลี่ยนลิ้งจาก main.scss เป็น main.css
